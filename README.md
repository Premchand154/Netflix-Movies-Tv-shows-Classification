# Netflix-Movies-Tv-shows-Classification
This project performs Exploratory Data Analysis (EDA) on Netflix’s dataset to uncover insights about movies and TV shows available on the platform. The analysis focuses on content distribution, trends over time, genres, ratings, and country-wise availability.

## Dataset

* **Source:** Netflix Movies and TV Shows dataset
* **Format:** CSV
* **Key Columns:**

  * `title`
  * `type` (Movie / TV Show)
  * `release_year`
  * `rating`
  * `duration`
  * `country`
  * `listed_in` (genre)

---

## Tools & Libraries Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## Key Analysis Performed

* Distribution of Movies vs TV Shows
* Content added over the years
* Most common ratings
* Top genres on Netflix
* Country-wise content distribution
* Duration analysis for movies and TV shows

---

## Insights

* Netflix has more **movies than TV shows**
* Significant growth in content after **2015**
* **TV-MA** is the most common rating
* The **United States** contributes the highest amount of content
* Drama and International content dominate the platform

## How to Run the Project


```bash
cd Netflix_EDA
```

3. Open Jupyter Notebook:


4. Run `Netflix_EDA.ipynb`
